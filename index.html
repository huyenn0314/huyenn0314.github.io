Selected Projects

Fine-tuned foundation model for cytology images classification

Implement and experiment with foundation models for cytology images classification and analysis for ovarian cancer screening and diagnosis. The project involves building annotation and visualization tool for Whole Slide Image and Histopathology Foundation Model knowledge for deep image features extraction.

Automatic density estimation of nanoparticles in light microscope images

Utilizing deep neural network for automatic, efficient particle density estimation in a novel light microscope device. The device, called PRAM, is innovated by Nano Sensor Group, UIUC, with the aim to create a portable poitnt-of-care solution for disease screening.

Fine-grained contrastive learning with textual priors

We applied signal processing, signal sparsification to construct text priors in fine-grained contrastive learning. The framework aims to establish an explainable ViT model for fine-grained image patch and text alignment

View Project PDF

A Comparative Study of Flow-Based and Diffusion-Based 3D Molecule Generation Methods

We conducted a comparative study of two state-of-the-art generative frameworks: FlowMol, a continuous- time flow-based model, and GeoLDM, a diffusion-based. This project revealed distinct trade-offs between the two approaches; provide practical insights into the strengths and limitations of flow- and diffusion-based models for molecular design.
